Authoring modules

What are user modules and how to use?

Modules help creating custom tags that can be used with custom paramaters in articles content or in template files to provide additional functionality to the web site. Any PHP or JAVASCRIPT code can be run inside a custom tags.

...

Users can create custom modules build in php and javascript in order to add new functionality to the website.

All modules are saved in the ./sites/default/modules directory.

Naming system

User modules names should not contain special characters or spaces. Underscores and camel case can be used. The main php file containing the module's code should be placed in the modules directory, with the inc extension, eg. ./sides/default/modules/myFirstModule.inc is a valid module file name.
